API Access

Programmatically start searches and retrieve leads

1. Start a Search

POST /api/search
Content-Type: application/json

{
  "industry": "SaaS",
  "location": "New York",
  "max_companies": 30,
  "max_pages": 5,
  "min_score": 40
}

// Response: { "job_id": 1, "status": "pending" }

2. Check Job Status

GET /api/jobs/{job_id}

// Response:
{
  "id": 1,
  "status": "running",
  "current_stage": "crawling",
  "total_companies": 30,
  "processed_companies": 12
}

3. Retrieve Leads

GET /api/leads?page=1&min_score=60&industry=SaaS&city=New+York

// Response:
{
  "total": 25,
  "page": 1,
  "leads": [
    {
      "company": "Acme Corp",
      "email": "info@acme.com",
      "lead_score": 85,
      "email_valid": true
    }
  ]
}

4. Download Excel Export

GET /export
// Returns: leads.xlsx file download

Quick Example (cURL)

# Start a search
curl -X POST http://localhost:8000/api/search \
  -H "Content-Type: application/json" \
  -d '{"industry":"Fintech","location":"London"}'

# Poll until complete
curl http://localhost:8000/api/jobs/1

# Fetch leads
curl http://localhost:8000/api/leads?min_score=50
